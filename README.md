# AviUtl2 プラグイン/スクリプト一覧

AviUtl2で利用可能なプラグインとスクリプトの一覧を提供するウェブアプリケーションです。

## 特徴

- モダンなレスポンシブデザイン
- 高速なページ読み込み（コード分割、遅延読み込み）
- SEO最適化（OGP、構造化データ、sitemap.xml）
- パフォーマンス最適化（クリティカルCSS、フォント最適化）
- ダークテーマ対応

## 開発環境のセットアップ

### 前提条件

- Node.js 16以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
https://github.com/lineside0418/AviUtl2_Plugins.git
cd AviUtl2_Plugins

# 依存関係をインストール
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

開発サーバーは [http://localhost:5173](http://localhost:5173) で起動します。

## ビルド

本番用ビルドを作成するには、以下のコマンドを実行します：

```bash
npm run build
```

ビルドは `dist` ディレクトリに出力されます。

## デプロイ

GitHub Pages を使用してデプロイする場合：

1. リポジトリの設定で GitHub Pages を有効にします
2. ソースを `gh-pages` ブランチまたは `main` ブランチの `/docs` フォルダに設定します
3. ビルドコマンドを実行して変更をプッシュします

## プロジェクト構成

```
src/
├── components/     # 再利用可能なUIコンポーネント
├── pages/         # ページコンポーネント
├── styles/        # スタイルシート
│   ├── style.css  # メインスタイルシート
│   └── optimized.css  # 最適化されたスタイルシート
├── App.jsx        # メインアプリケーションコンポーネント
└── main.jsx       # アプリケーションのエントリーポイント
```

## パフォーマンス最適化

以下の最適化が実装されています：

- クリティカルCSSのインライン化
- 非クリティカルCSSの非同期読み込み
- フォントの最適化
- 画像の遅延読み込み
- ルートベースのコード分割
